Adds VAT field to WooCommerce checkout page.

== Description ==

Adds VAT field to WooCommerce checkout and customer account.

Adds VAT filed to WooCommerce REST (wc/v1 and wc/v2) orders endpoint.

Customizable position in checkout and customer account form.

Installation:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to WooCommerce -> Settings -> VAT Field -> Enable -> Save
